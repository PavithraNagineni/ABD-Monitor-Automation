# ⚡ System Monitor - GUI Dashboard

A real-time system monitoring dashboard built with Python, tkinter, and psutil.

---

## 📊 Features
- **CPU Usage** — Live percentage + frequency + core count
- **Memory Usage** — Used/Total in GB with progress bar
- **Disk Usage** — Storage consumption with progress bar
- **Network Monitor** — Total sent/received + live speed (KB/s)
- **Top Processes** — Live top 7 CPU-consuming processes
- **Auto-refresh** every 1.5 seconds

---

## 🚀 Setup & Run

### 1. Install dependencies
```bash
pip install psutil
```

### 2. Run the dashboard
```bash
python system_monitor.py
```

---

## 🛠 Tech Stack
- Python 3.x
- tkinter (built-in GUI library)
- psutil (system stats)
